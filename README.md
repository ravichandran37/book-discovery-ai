# discover.
### An AI-Powered Digital Librarian

A full-stack book discovery application built with **Django**, **Gemini 2.5 Flash**, and **ChromaDB**. 

## Features
* **Conversational Memory**: The AI remembers previous searches.
* **Semantic Search**: Uses vector embeddings to find books by "vibe" rather than just keywords.
* **Minimalist UI**: Responsive chat interface designed with Tailwind CSS.

## Tech Stack
* Backend: Python / Django
* AI: Google Gemini API
* Database: ChromaDB (Vector Store)
* Frontend: Tailwind CSS / JavaScript
